Fonctionnalités

Ce que fait l'application

Un outil simple et élégant pour générer des palettes de couleurs aléatoires, idéal pour les designers et développeurs en quête d'inspiration.

🎲
Génération aléatoire

Un clic sur le bouton génère instantanément 5 couleurs aléatoires parmi plus de 16 millions de combinaisons possibles.

📋
Copie en un clic

Cliquez sur le bouton copier ou directement sur la couleur pour copier le code HEX dans le presse-papiers instantanément.

✅
Feedback visuel

L'icône de copie se transforme en coche verte pendant 1,5 secondes pour confirmer que le HEX a bien été copié.

🎨
Affichage des codes HEX

Chaque couleur affiche son code HEX en clair sous la couleur pour une utilisation directe dans du code ou des logiciels de design.

✨
Animations soignées

Effets hover avec élévation des cartes, transitions fluides et soulignement animé du titre pour une expérience agréable.

📱
Responsive design

La grille de couleurs s'adapte automatiquement à toutes les tailles d'écran grâce à CSS Grid avec auto-fit.

Apprentissages

Ce que j'ai appris

📋

Clipboard API

Utilisation de navigator.clipboard.writeText() pour copier du texte dans le presse-papiers avec gestion des promesses .then() et .catch().

🎲

Génération aléatoire

Construction d'une couleur HEX aléatoire caractère par caractère avec Math.random() et un tableau de 16 caractères hexadécimaux.

🎯

Event delegation

Utilisation de la délégation d'événements sur le conteneur parent pour gérer les clics sur tous les boutons de copie avec un seul listener.

🔗

previousElementSibling

Navigation dans le DOM avec previousElementSibling et nextElementSibling pour accéder aux éléments adjacents sans sélecteurs complexes.
