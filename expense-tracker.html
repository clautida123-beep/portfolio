Fonctionnalités

Ce que fait l'application

Un tracker de dépenses complet qui calcule automatiquement le solde, les revenus et les dépenses à chaque ajout ou suppression de transaction.

➕
Ajout de transactions

Formulaire simple avec description et montant. Les valeurs positives sont des revenus, les négatives des dépenses.

🗑️
Suppression de transactions

Bouton de suppression discret qui apparaît au survol de chaque transaction pour une interface propre.

💰
Calcul automatique du solde

Le solde total, les revenus et les dépenses se mettent à jour instantanément après chaque action.

💾
Sauvegarde automatique

Les transactions sont sauvegardées dans le localStorage — elles persistent même après fermeture du navigateur.

🎨
Code couleur visuel

Les revenus sont affichés en vert et les dépenses en rouge, avec une bordure latérale colorée sur chaque transaction.

📱
Responsive design

Interface adaptée aux mobiles et tablettes avec grille flexible et mise en page verticale sur petits écrans.

Apprentissages

Ce que j'ai appris

💾

localStorage

Persistance des données côté navigateur avec JSON.stringify et JSON.parse pour sauvegarder et récupérer les transactions entre les sessions.

🔢

Array.reduce()

Utilisation de reduce() pour calculer le solde total, les revenus et les dépenses en parcourant le tableau des transactions.

🎯

Intl.NumberFormat

Formatage professionnel des montants en devise USD grâce à l'API native Intl.NumberFormat pour un affichage clair et standardisé.

🎬

Animations CSS

Animation slideIn sur chaque nouvelle transaction et effets hover sur les éléments interactifs pour une expérience fluide.
